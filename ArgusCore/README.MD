# ArgusCore

ArgusCore is a modular operational monitoring platform built with Python and PostgreSQL.

## Features

- Continuous monitoring
- Brute force detection
- Port scan detection
- Critical event analysis
- Runtime debug mode
- Monitoring console
- Password protection
- Lockdown system

## Technologies

- Python
- PostgreSQL
- Psycopg2
- Threading
- Django
